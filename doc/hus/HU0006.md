## 1. Especificação da História de Usuário

- **Como:** Usuário autenticado no site
- **Quero:** Comentar em uma postagem
- **Para:** Deixar meu comentário público

<br>

## 2. Cenários

### **2.1. Comentar em Postagem**

- **DADO** Que sou usuário autenticado
- **QUANDO** Escrevo um comentário no campo de texto
- **E** Clico no botão de enviar
- **ENTÃO** O sistema salva as informações e exibe a mensagem de sucesso

<br>

## 3. Critérios de Aceitação:

- [x] **3.1.** O comentário deve ser vinculado a postagem referente.

- [x] **3.2.** Uma mensagem de confirmação deve ser exibida.

<br>
